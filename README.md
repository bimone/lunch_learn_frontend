### Getting started (Command Line)
**Cloning the repo:** </br>
`git clone https://github.com/bimone/lunch_learn_frontend.git`</br>
`cd lunch_learn_frontend`

**Install dependencies from the root of the project:** </br>
`yarn install` or `npm install`
